# RTT Real-Time Translate (MVP)

Прототип приложения для **перевода текста с экрана в реальном времени**.

## Что делает текущая версия

1. Делает кадры с выбранной области экрана.
2. Распознаёт текст (OCR).
3. Переводит текст на целевой язык.
4. Показывает перевод в отдельном окне.
5. Разбивает результат на блоки, чтобы текст не превращался в «месиво».

> В этой версии оригинальный текст на исходном экране не удаляется, а перевод показывается в отдельном окне.

## Архитектура MVP

- `ScreenCaptureService` — захват области экрана.
- `OCRService` — распознавание текста через Tesseract.
- `TranslationService` — перевод через Google Translate (библиотека `deep-translator`).
- `TextSegmenter` — группировка и разделение OCR-блоков.
- `TranslatorApp` — UI и цикл обновления.

## Установка

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Также нужен установленный движок **Tesseract OCR**:
- Linux: `sudo apt-get install tesseract-ocr`
- Windows: установить с официального инсталлятора и добавить в PATH.

## Запуск

```bash
python app.py
```

## Управление

- Поля `OCR язык`, `Из языка`, `В язык`.
- Кнопка **Запустить** — старт обработки.
- Кнопка **Остановить** — остановка.
- Поле **Интервал (сек)** — частота обновления.

## Ограничения текущей версии

- Может переводить один и тот же текст повторно (есть простая дедупликация, но не идеальная).
- Качество OCR зависит от шрифта, качества картинки и языка.
- Перевод зависит от сетевого доступа.
